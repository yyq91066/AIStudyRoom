# 微信小程序快速上手指引

## ✅ 完成！项目已创建成功

所有必需的文件已经创建完毕，你现在可以直接导入到微信开发者工具中。

## 📋 接下来的步骤

### 第一步：准备图标资源 (5 分钟)

你需要在 `miniprogram/images/` 文件夹中放置以下 6 个图标:

1. **logo.png** (160x160px) - Logo 图片
2. **wechat.png** (48x48px) - 微信图标
3. **chat.png** (48x48px) - 聊天图标 (未选中)
4. **chat-active.png** (48x48px) - 聊天图标 (选中)
5. **profile.png** (48x48px) - 个人中心图标 (未选中)
6. **profile-active.png** (48x48px) - 个人中心图标 (选中)

**临时方案**: 如果暂时没有图标，可以用画图工具随便画几个纯色方块代替，或者从网上下载免费图标。

### 第二步：修改 AppID (2 分钟)

1. 打开 `project.config.json` 文件
2. 找到 `"appid": "wxXXXXXXXXXXXXXXX"`
3. 替换成你的小程序 AppID (如果没有，先跳过，在开发者工具中选择"测试号")

### 第三步：导入到微信开发者工具 (3 分钟)

1. 打开微信开发者工具
2. 选择"导入项目"
3. 项目目录选择：`D:\developData\bishe\aidemo\miniprogram`
4. AppID 填写你的小程序 ID (或使用测试号)
5. 点击"确定"

### 第四步：配置开发环境 (1 分钟)

1. 在微信开发者工具中，点击右上角的"详情"
2. 勾选"不校验合法域名、web-view(业务域名)、TLS 版本以及 HTTPS 证书"
3. 这样可以避免开发阶段的域名限制

### 第五步：启动后端服务 (重要!)

确保你的 Spring Boot 后端服务运行在 `http://localhost:8080`

检查后端是否启动:
- 访问 http://localhost:8080 应该能看到后端响应

### 第六步：编译运行

1. 在微信开发者工具中点击"编译"按钮 (或按 Ctrl+B)
2. 等待编译完成
3. 模拟器中将显示小程序界面

## 🔧 常见问题排查

### 问题 1: 编译后显示空白页面
**解决方案**: 
- 检查是否缺少图标文件
- 打开调试控制台查看错误信息

### 问题 2: 登录请求失败
**解决方案**:
- 确认后端服务已启动 (访问 localhost:8080)
- 检查 http.js 中的 API_BASE_URL 是否正确
- 确认已关闭域名校验

### 问题 3: 验证码发送失败
**解决方案**:
- 检查后端邮件服务配置
- 查看后端控制台的错误日志

### 问题 4: tabBar 图标不显示
**解决方案**:
- 检查 images 文件夹中是否有对应的图标文件
- 图标文件名是否正确 (区分大小写)

## 📱 真机预览

如果想在自己的手机上测试:

1. 点击微信开发者工具顶部的"预览"按钮
2. 使用微信扫描二维码
3. 小程序将在手机上打开

注意：真机预览需要将 localhost 改为你电脑的局域网 IP 地址
